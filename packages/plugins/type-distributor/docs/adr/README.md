# Architecture decision records

These ADRs capture the durable decisions for `type-distributor`.

They follow the [Michael Nygard](https://github.com/joelparkerhenderson/architecture-decision-record/tree/main/locales/en/templates/decision-record-template-by-michael-nygard) structure:

- Title
- Status
- Context
- Decision
- Consequences
